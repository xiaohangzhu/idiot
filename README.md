# 木头人

## 简介
   一款可以禁止用户手机操作，拦截用户操作事件的Android平台应用。
   
## 适用人群
   1.学习，考研究等情况下控制不住自己玩手机
   
   2.睡觉前玩手机停不下来
   
   3.朋友聚会，告别手机，敞开心扉
